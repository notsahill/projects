Install

  1. Open the .dmg → drag DeskPet to Applications.
  2. First launch is blocked (unsigned). In Terminal, run: 
    xattr -dr com.apple.quarantine /Applications/DeskPet.app
    then double-click DeskPet. (Or right-click → Open if you only see "unidentified developer".)
  3. Lives in the menubar 🐾. For typing/scroll reactions: System Settings → Privacy & Security → Accessibility → enable DeskPet → relaunch.
