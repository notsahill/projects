two minutes

getting it home

1
Open the .dmg and drag DeskPet into Applications.
2
Unsigned build, so clear the quarantine flag once in Terminal:
xattr -dr com.apple.quarantine /Applications/DeskPet.app
…then double-click. (Or right-click → Open if you only see "unidentified developer".)
3
Lives in the menubar 🐾. For typing reactions: System Settings → Privacy & Security → Accessibility → enable DeskPet → relaunch.
⊞
Windows: run the .exe installer. If SmartScreen warns, click More info → Run anyway. Typing reactions work with no extra permission. Lives in the system tray 🐾.
4
Open Settings from the tray to pick colors, set reminder timers, and one-click connect your AI IDEs.
